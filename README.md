# CS200
Pokémon Tower RPG
A text-based, turn-based RPG inspired by the classic Pokémon battle system, built entirely in Java for UW–Madison CS 200 Project 2. Traverse an 8-floor tower, battle diverse Pokémon, manage your team, catch fallen foes, and challenge the ultimate boss—Mewtwo!

Features
8-floor progressive difficulty tower
Type-based battle system (Fire, Water, Grass, Electric, Ground) with elemental advantages
Team management & item bag system (switch, store, and deploy Pokémon)
Catch defeated wild Pokémon to expand your roster
Level-up mechanics with stat growth after defeating strong foes
Epic boss battle with Mewtwo – transforms at half HP for a dramatic second phase!
How to Run
bash
12
javac Main.java
java Main
Requires Java 8 or higher.

宝可梦塔文字冒险游戏
这是一款基于经典宝可梦对战系统开发的纯文字回合制 RPG 游戏，使用 Java 语言完整实现，作为威斯康星大学麦迪逊分校（UW–Madison）CS 200 课程项目二提交。玩家将挑战一座 8 层高塔，与各系宝可梦对战，管理队伍，捕捉击败的敌人，并最终迎战终极 Boss——超梦（Mewtwo）！

特色功能
8 层递增难度的高塔挑战
基于属性克制的战斗系统（火、水、草、电、地面）
队伍与背包管理系统（切换、存放、部署宝可梦）
捕捉击败的野生宝可梦，扩充你的队伍
击败强敌后可升级，提升生命值与攻击力
与超梦的史诗级 Boss 战——血量过半时触发二阶段变身！
