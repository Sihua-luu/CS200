# CS200 - Pokémon Tower RPG

A text-based, turn-based RPG inspired by the classic Pokémon battle system, built entirely in Java for UW–Madison CS 200 Project 2. Traverse an 8-floor tower, battle diverse Pokémon, manage your team, catch fallen foes, and challenge the ultimate boss—Mewtwo!

## Features
- **8-floor progressive difficulty tower** - Each floor presents increasingly difficult challenges
- **Type-based battle system** - Includes Fire, Water, Grass, Electric, and Ground types with elemental advantages
- **Team management & item bag system** - Switch, store, and deploy Pokémon strategically
- **Catch defeated wild Pokémon** - Expand your roster by capturing defeated foes
- **Level-up mechanics** - Gain stat growth after defeating strong opponents
- **Epic boss battle** - Face Mewtwo with a dramatic second-phase transformation at half HP!

## How to Run
```bash
javac Main.java
java Main
```
# 宝可梦塔文字冒险游戏

一款基于经典宝可梦对战系统开发的纯文字回合制 RPG 游戏，使用 Java 语言完整实现，作为威斯康星大学麦迪逊分校（UW–Madison）CS 200 课程项目二提交。玩家将挑战一座 8 层高塔，与各系宝可梦对战，管理队伍，捕捉击败的敌人，并最终迎战终极 Boss——超梦（Mewtwo）！

## 特色功能
- **8 层递增难度高塔挑战** - 每层难度逐渐提升，带来更具挑战性的冒险
- **基于属性克制的战斗系统** - 包含火、水、草、电、地面五种属性，遵循经典克制关系
- **队伍与背包管理系统** - 可切换、存放和部署宝可梦，进行策略性队伍管理
- **捕捉击败的野生宝可梦** - 击败野生宝可梦后有机会捕捉，扩充你的收藏
- **升级成长机制** - 击败强敌后可升级，提升生命值与攻击力
- **史诗级 Boss 战** - 与超梦的最终对决，血量过半时触发震撼的二阶段变身！

## 运行方法
```bash
javac Main.java
java Main
```
